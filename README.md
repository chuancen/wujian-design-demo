# 五健体检平台设计 Demo

这是儿童青少年“五健”体检管理平台的 UI/UX 设计截图 demo。

## 在线/本地查看

打开 `index.html`，可在手机端浏览四套设计方案：

- 方案A：省级政务医疗平台
- 方案B：实时指挥中心产品
- 方案C：儿童健康服务品牌产品
- 方案D：临床采集效率工具

每套方案包含 32 个页面级截图，共 128 张。

## 文件说明

- `index.html`：移动端优先的设计查看页。
- `screenshots-v2/`：新版生产设计截图。
- `SCREENSHOTS_V2.md`：截图清单与说明。
- `PRD.md`：需求分析与页面规划。

## 页面编号

图片命名格式为：

```text
P编号-页面名称.png
```

例如：

- `P01-登录-角色入口.png`
- `P15-单人采集录入.png`
- `P29-区域统计驾驶舱.png`
